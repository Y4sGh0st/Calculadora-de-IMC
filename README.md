CALCULADORA DE IMC

  DESCRIÇÃO
 
Este projeto foi desenvolvido como parte da disciplina de Desenvolvimento de Sistemas.
Ele consiste em uma calculadora de IMC (Índice de Massa Corporal) feita em Python, com o objetivo de praticar conceitos básicos de programação, como entrada de dados, operações matemáticas e estruturas condicionais.

O sistema permite que o usuário informe seu peso e altura, calcula automaticamente o IMC e exibe o resultado junto com a classificação (abaixo do peso, peso normal, sobrepeso ou obesidade), de acordo com padrões utilizados na área da saúde.


 FUNCIONALIDADES
 
- Entrada de peso e altura pelo usuário

- Cálculo automático do IMC

- Exibição do resultado com duas casas decimais

- Classificação do IMC:

Abaixo do peso

Peso normal

Sobrepeso

Obesidade


 COMO EXECUTAR O PROJETO
 
1. Instale o Python (versão 3 ou superior).

2. Salve o código em um arquivo chamado imc.py.

3. Abra o terminal na pasta do arquivo.

4. Execute:
python imc.py
ou
python3 imc.py

5. Digite seu peso e altura quando solicitado.
